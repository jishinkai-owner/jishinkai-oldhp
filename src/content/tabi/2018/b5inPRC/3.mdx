莫高窟数字展示中心
<br/>
<br/>
莫高窟数字展示中心は莫高窟の資料館のようなもので、
<br/>
莫高窟を観光する場合はここで必ず2本立ての映像を見ないといけないルール
<br/>
<br/>
腹が減ってたので、まずは館内のカフェテリアで昼飯を食った
<br/>
<br/>
<a data-flickr-embed="true"  href="https://www.flickr.com/gp/96951391@N03/ge8C7Z" title="敦煌_180330_0118"><img src="https://farm1.staticflickr.com/878/39300375540_5c9aedfbb4.jpg" width="500" height="375" alt="敦煌_180330_0118"/></a>

昼食のカレーセット
<br/>
<br/>
筆者はカレーセットを頼んだが、味は日本のレトルトそのもので、
<br/>
茶碗蒸しもあまり美味しくなかった
<br/>
伊東が頼んだものは日本の牛丼とほぼ同じものだった
<br/>
西域風のものを期待していただけにガッカリ
<br/>
<br/>
そうこうしてる内に上映時間に
<br/>
前半は莫高窟誕生の経緯を描いた歴史映画的なもの
<br/>
後半は石窟内を3Dで再現した映像だった
<br/>
<br/>
筆者は前半の上映中は寝ていたのでよくわからないけれど、
<br/>
伊東と山﨑は絶賛してたから面白いんでしょう
<br/>
後半は見応えあったと思う
