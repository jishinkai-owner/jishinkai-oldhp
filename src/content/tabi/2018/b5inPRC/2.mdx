豆腐脑 たったの1元
<br/>
<br/>
列車の発射時刻が早いので、7時くらいに起床
<br/>
中国は全土の時間帯が同じなため、西安の朝は暗い
<br/>
<br/>
まずは宿の近くで朝食
<br/>
豆腐脑は中国の定番朝ごはんで、地域によって名前も味も違うらしい
<br/>
西安のものはやや辛めの味
<br/>

<br/>
朝マックかな
<br/>
<br/>
そのままタクシーで西安駅へ
<br/>

乗車するまで少々時間がかかった
