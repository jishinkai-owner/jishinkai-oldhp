4号線を北上して「入水鍾乳洞」に向かいました。<br/>
この鍾乳洞は、スゴいです。<br/>
短パン、ヘッドライト、草履、着替えは必須でしょう。<br/>
「ちょっとした探検気分」どころじゃないです。<br/>
「これ、ちゃんと出られるかな」って思うくらい厳しいです。<br/>
<br/>
入口の時点でなかなか狭く、厳しいものがあるのですが、帰りにはそれすら楽に思えました。<br/>
<br/>
足を水に入れると、冷ッたい！<br/>
泣きそうになるくらい冷たい。<br/>
14度くらいだったかな。<br/>
冷たさは、そのうち慣れます。<br/>
<br/>
後半は、明りが無いです。<br/>
完全に闇です。<br/>
手元の明りだけが頼りです。<br/>
厳しいすり抜けや、四つん這いも必要です。<br/>
…明りが切れたら死ねる。<br/>
<br/>
出たころには、Tシャツも短パンもぐっしょり濡れていました。<br/>
<br/>
<br/>
夜は、郡山の近くの定食屋で最後の夕食です。<br/>
うまい・・・！うまいぞ！<br/>
量も多く、大満足でした。<br/>
