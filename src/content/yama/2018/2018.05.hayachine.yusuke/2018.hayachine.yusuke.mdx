title: 早池峰山　YUSUKE編

<p>　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　<span style="font-size: 12pt;">５月２６日、２７日早池峰山（岩手県）</span></p>
<p><span style="font-size: 12pt;">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 車は佐々木さん、田村さん、井野場さん、梨本氏、私の５人でした。初日は天気も良く、気持ちよかったですね。</span></p>
<p><span style="font-size: 12pt;">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;&nbsp; 焼肉店でランチを食べました。私は本場の盛岡冷麺を食べましたが非常に美味しく各々大満足でした。やっぱ本場は違いますね。</span></p>

<p><span style="font-size: 12pt;">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp;&nbsp; その他色々感じられる素晴らしいトコでした。機会があれば行ってみることをお勧めします。写真は記念館の近くにある食事処兼土産店の山猫店です。</span></p>
