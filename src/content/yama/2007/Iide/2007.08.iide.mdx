<p class="MsoNormal" style="text-indent:12pt; mso-char-indent-count:1.0" align="left">
<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">仙台駅で</span>

<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">の東北本線に乗車。途中の駅で線路に人が落ちたので電車が緊急停車した。この影響で電車が５分ほど遅れる。郡山駅に</span>
<span lang="EN-US">9:29</span>
<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">に着き</span>

<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">の会津若松駅行きの磐越西線に乗り換える。</span>
<span lang="EN-US">10:50</span>
<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">会津若松駅に着く。</span>

<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">快速あがの</span>
<span lang="EN-US">1</span>
<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">号に乗り換え</span>

<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">野沢駅で下車。野沢駅の近くで南の祖父母に昼食をご馳走になりタクシーで弥平四郎登山口へ向かう。</span>
<span lang="EN-US">14:15</span>
<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">弥平四郎登山口。登山口からすぐにくだりにはいり奥川にでる。川には丸太二本に板をわたした橋がかかっている。橋なしでも増水してなければ楽にわたれる。ただし増水時は危険。橋は石で両岸から止めてあるだけなので洪水になれば簡単に流されるだろう。</span>
</p>

<p class="MsoNormal" style="text-indent:12pt; mso-char-indent-count:1.0" align="left">

<span lang="EN-US">20</span>
<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">分後の</span>

<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">に祓川山荘に着いた。登山道はしっかり刈払いされている。今日はここに宿泊。暗くなってから焚き火をしようという話になり枝を集めるがしけっていて燃えない。雨が降ってきたので小屋の中に入る。雨は夜の間強く降ったり弱くなったりを繰り返し降り続く。</span>

</b><b style='mso-bidi-font-weight:normal'>
<span style='font-size:14.0pt;font-family:"ＭＳ 明朝","serif";mso-ascii-font-family:Century;mso-hansi-font-family:Century'>日　雨</span>
</b></p>

<p class="MsoNormal" style="text-indent:12pt; mso-char-indent-count:1.0" align="left">
<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">朝から雨。天図を</span>
<span lang="EN-US">3</span>
<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">回取る。天気図を見ると発達した低気圧の寒冷前線が通過中で天候が回復する兆しはないので祓川山荘に停滞することになる。奥川の橋が流されていないか心配になり見に行くが川はほとんど増水しておらず橋も無事だった。</span>
<span lang="EN-US">1</span>
<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">日中トランプをして過ごす。</span>
</p>

<p class="MsoNormal" style="text-indent:12pt; mso-char-indent-count:1.0" align="left">
<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">夕食はタラコとイカスミのパスタ。イカスミはおいしいがイカスミの処理がめんどうで黒いイカスミはおちにくく山行中ずっと黒い色がコッヘルに残った。</span>
</p>

<p class="MsoNormal" style align="left">
<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">よってあまり登山の食事としてイカスミは勧められない。</span>

<p align="left" class="MsoNormal">&nbsp;</p>
<p class="MsoNormal" style align="left"><b style='mso-bidi-font-weight:normal'>
<span lang="EN-US" style='font-size:14.0pt'>8</span>
<span style='font-size:14.0pt;font-family:"ＭＳ 明朝","serif";mso-ascii-font-family:Century;mso-hansi-font-family:Century'>月</span>

<p class="MsoNormal" style="text-indent:12pt; mso-char-indent-count:1.0" align="left">
<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">寒冷前線は完全に抜け切っていなかったが</span>
<span lang="EN-US">22</span>
<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">日よりも天候は回復しそうだったので出発することにする。天気は曇り</span>
<span lang="EN-US">6:30</span>
<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">祓川山荘を出発する。松平峠まではブナの樹林帯で登山道は特に危険な箇所はない。</span>
<span lang="EN-US">7:15</span>
<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">から</span>
<span lang="EN-US">7:35</span>
<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">と</span>
<span lang="EN-US">8:00</span>
<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">から</span>
<span lang="EN-US">8:05</span>
<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">に休憩をとる。松平峠には</span>
<span lang="EN-US">8:30</span>
<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">に着き</span>
<span lang="EN-US">8:45</span>
<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">まで休憩した。松平峠までの水場は</span>
<span lang="EN-US">2</span>
<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">箇所である。ひとつは祓川山荘から近い祓川山荘の水場の源流と思われるものでコンクリの箱に水が流れ落ちている。もうひとつは十森の水場で沢水である。水量はちょろちょろ。雨が少ないと枯れそう。松平峠は木に看板がかかっているのですぐに分かる。松平峠を過ぎたあたりから青空が見え始め日差しがでできた。松平峠からは尾根で松平峠から猪鼻までは登りが急で登山道の脇が崖になっている部分がある。また登山道の地面は土ではなく花崗岩砂で登山道上に岩が多い。植生は人の背丈ほどのナナカマドやアザミの樹林帯</span>
</p>

<p class="MsoNormal" style align="left">
<span lang="EN-US">9:25</span>
<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">猪鼻につく。</span>

<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">まで休憩。猪鼻には水場があるがロープ伝いに下りる必要があり、水量も少なく飲料水としては適さない。</span>

<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">まで休憩。</span>
<span lang="EN-US">12:55</span>
<span style="font-family:'ＭＳ 明朝',serif; mso-ascii-font-family:Century;mso-hansi-font-family:Century">種蒔山に着き、</span>
