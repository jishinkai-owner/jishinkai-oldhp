ネギを両手に気合充分なSL塚原氏
<br/><br/>
8月14日、我々は仙台から遠くはなれた松本にいた
<br/>
目指すは北アルプス、槍ヶ岳の頂
<br/>
日も落ちた頃、翌日より始まる山行について激論を繰り広げていた
<br/>
台風が我が国に迫っており、山行中に危険に晒される可能性があったのだ
<br/>
<br/>
<a data-flickr-embed="true"  href="https://www.flickr.com/gp/96951391@N03/1kW54G" title="P8140006"><img src="https://c7.staticflickr.com/9/8573/29028283990_75acbca221.jpg" width="500" height="375" alt="P8140006"/></a><script async src="//embedr.flickr.com/assets/client-code.js" charset="utf-8"></script>
<br/>
激論を繰り広げる自親会員達、疲れの色を見せる者もチラホラ
<br/>
<br/>
長きにわたる議論の末、我々は日程を短縮するという結論に達した
<br/>
最短4泊5日で行う予定だった山行を、2泊3日へと変更したのである
<br/>
<br/>
<a data-flickr-embed="true"  href="https://www.flickr.com/gp/96951391@N03/a6Jy88" title="IMG_2534"><img src="https://c5.staticflickr.com/9/8812/29377751540_2d0c74982e.jpg" width="375" height="500" alt="IMG_2534"/></a><script async src="//embedr.flickr.com/assets/client-code.js" charset="utf-8"></script>
<br/>
それぞれの思いを胸に眠りに就く自親会員達
<br/>
<br/>
そうして、翌日からの山行に備えて眠りについたのであった
