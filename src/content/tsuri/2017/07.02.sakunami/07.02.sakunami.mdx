作並(2017/07/02)
</span>
<br/>
<br/>
<br/>
<br/>
<span style="font-size: 12pt;">
作並渓流釣り場でニジマス釣りをしたんで、報告するぜ！
<br/>
<br/>
今回の参加者は原田さん(B3)、井上(B5)、加藤(B5)
<br/>
<br/>
髙橋も来る筈だったが、前日の飲み会の影響で来られず
<br/>
また今度一緒に釣りしような
<br/>
<br/>
<a data-flickr-embed="true"  href="https://www.flickr.com/gp/96951391@N03/2955FM" title="IMG_5403"><img src="https://farm6.staticflickr.com/5785/29832110794_27bb5c863a.jpg" width="500" height="333" alt="IMG_5403"/></a>
<br/>
<br/>
元々は閖上で釣る予定だったんだが、現地に着いてみると近くに釣具屋が無かったんで、
<br/>
急遽作並に目的地を変更
<br/>
<br/>
写真は撮り損ねちまったが、閖上は良いところだったぜ
<br/>
<br/>
一旦部室に寄って、道具を準備をしたら作並に向かって出発だ！
<br/>
<br/>
オラオラと運転してたら正午前に到着したぜ
<br/>
<br/>
2時間\2100のニジマス餌釣り券を買い、餌にイクラを購入したら釣り開始だ！
